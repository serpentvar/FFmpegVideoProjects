# MP4 Utilities

## Scripts

- `mp4_h264_nvenc_14000k.bat`: Encode with NVIDIA NVENC at 14000k.
- `mp4_h264_nvenc_1080p30_fast.bat`: Encode with NVIDIA NVENC using a fast 1080p30-friendly preset.
- `mp4_libx264_14000k.bat`: Encode with libx264 at 14000k.
- `mp4_libx264_8000k.bat`: Encode with libx264 at 8000k.

Output filenames include the timestamp and bitrate where the source script supports it.
