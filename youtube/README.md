# YouTube Scripts

Drag MP4 files onto one of these scripts to create YouTube-oriented MP4 outputs.

## Scripts

- `mp4_youtube_h264_nvenc.bat`: NVIDIA NVENC encode at 14000k with two-pass settings.
- `mp4_youtube_h264_nvenc_4k.bat`: NVIDIA NVENC encode for higher-resolution source material.
- `mp4_youtube_libx264.bat`: CPU/libx264 encode for systems without NVIDIA NVENC.

Output files are written next to the source videos.
