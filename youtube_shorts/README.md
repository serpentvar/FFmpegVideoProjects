# YouTube Shorts Scripts

Drag videos onto these scripts to create 1080x1920 Shorts-ready files.

## Scripts

- `mp4_short_h264_nvenc.bat`: Crop to 1080x1920 and encode with NVIDIA NVENC.
- `mp4_short_libx264.bat`: Crop to 1080x1920 and encode with libx264.
- `mp4_short_libx264_padded.bat`: Fit the full video inside 1080x1920 with padding instead of cropping.
- `mp4_short_fullscreen_overlay.bat`: Stack a full-width top view with a cropped lower view.
- `mp4_short_h264_nvenc_APEX_Legend_HP_overlay.bat`: NVENC variant with an Apex Legends HP overlay crop.
- `mp4_short_libx264_APEX_Legend_HP_overlay.bat`: libx264 variant with an Apex Legends HP overlay crop.
